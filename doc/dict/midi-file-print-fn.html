[Function]
(midi-file-print midifile {keyword value}*)

Prints the contents of midifile to a stream and returns midifile as its value.

midi-file-print supports the following keyword arguments:

:stream {stream}
The stream to print midifile to. The default value is the standard output.
:track number
The MIDI track number to print. The default value is 0.

Examples:

Example 1. Printing the contents of a MIDI file to the standard output. 

(define (simp reps)
  (process repeat reps
           output (new midi :time (now)
                       :duration 1
                       :keynum (between 50 90))
           wait .5))

(events (list (new midi-program-change :time 0 :program 30)
              (simp 12))
        "test.mid" :timesig '(2 4) :tempo 120)
⇒ "test.mid"
