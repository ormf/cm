[Function]
(scale-max freq1 freq2 [scale])

Returns freq2 if it is greater than freq1 in scale, otherwise returns freq1. 

cm> (scale-max 60 'c4)
60
cm> (scale-min 'css4 'cs4)
css4
cm>
