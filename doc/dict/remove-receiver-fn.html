[Function]
(remove-receiver! [stream])

Removes the receiver hook associated with the optional stream. Returns no values. If stream is not specified then all receivers in all streams are removed. If stream is boolean true then the current "generic receiver" is removed, otherwise stream should be a stream object with an active receiver to remove.

Examples:

See set-receiver! documentation for examples of using remove-receiver!.

See also:

  • set-receiver! [Function]
  • receiver? [Function]
